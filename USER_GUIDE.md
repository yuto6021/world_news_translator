## 1. 初回起動と基本操作
- ホーム画面上部に検索バー、右上に4つのアイコン: 🎰ガチャ / 📖実績図鑑 / 🛍️ショップ / ⚙️設定。
- 下部タブ／またはスクロール領域でニュース、国別、地図ニュース、ゲーム、統計、プロフィールなどへ切替可能。
- 初回起動時に連続ログインカウンタが開始され、ストリーク実績の基礎になります。

## 2. ニュース閲覧 & 翻訳
- 記事カードをタップすると詳細画面へ。
- DeepLキーが設定されていれば高精度翻訳。未設定時は簡易辞書フォールバック。
- 英語原文と日本語訳を交互に読むことで語彙学習。今後のアップデートでサイドバイサイド表示予定。
- 長文は将来的に「要約を見る」ボタン（AI要約）で短縮表示予定。

## 3. 読書モード / アクセシビリティ設定 (⚙️)

### 設定項目詳細
設定画面（右上⚙️アイコン）では以下をカスタマイズ可能：

#### テーマ設定
- **ライトモード**: 明るい背景で日中の読書に最適
- **ダークモード**: 暗い背景で夜間の目の負担を軽減
- **システム設定に従う**: 端末のテーマ設定を自動反映（推奨）

#### 読書モード
- **フォントサイズ**: 小 / 中（標準）/ 大 / 特大から選択
- **行間**: 狭い / 普通（標準）/ 広い / とても広い
- **ディスレクシックフォント**: 読字障害対応の専用フォント（OpenDyslexic風）
- **ハイコントラスト**: 文字と背景の明度差を最大化

#### 翻訳設定
- **DeepL APIキー**: 高精度翻訳を有効化（オプション）
- **翻訳キャッシュクリア**: メモリ不足時に実行

#### 通知設定（今後実装予定）
- デイリーダイジェスト配信時刻
- ストリーク継続リマインダー
- 実績解除通知

### 推奨設定組み合わせ
| 用途 | フォント | 行間 | コントラスト | テーマ |
|------|---------|------|-------------|-------|
| 快適読書 | 中 | 広い | OFF | システム |
| 語学学習 | 大 | とても広い | OFF | ライト |
| 視覚サポート | 特大 | とても広い | ON | ダーク |
| ディスレクシア | 大（専用） | 広い | ON | ダーク |

## 4. お気に入り & タイムカプセル
- 記事詳細画面の「★」でお気に入り追加。統計タブに件数が反映。
- タイムカプセル: 後で読み返したい記事を保存して学習進捗を振り返る。ショップ拡張で保存枠増加予定。

## 5. 実績システムと図鑑 (📖)
- 達成条件：読書時間、連続ログイン、ゲームスコア、コメント数など。
- レア度：コモン / レア / エピック / レジェンダリー。レアほど演出が豪華 & 付与ポイント増。
- 図鑑では解除済み実績数と未解除の影表示。テスト用の実験実績（🧪）は開発中のみ統計画面から解除可能。

## 6. ガチャ & チャレンジ (🎰)
- 1日1回ガチャ（開発版では回数無制限）。レア度で難易度と報酬が変化。
- チャレンジ例：記事を指定数読む / コメント投稿 / お気に入り追加 / ゲームスコア達成 / クイズ高得点。
- 24時間以内に達成すると報酬（ポイント）＋実績進捗に寄与。
- ガチャ結果はスナックバーと演出オーバーレイで表示。

## 7. ショップ & テーマ適用 (🛍️)
- ポイントでテーマ・ペットアイテム・拡張アイテムを購入。
- テーマは購入 → 購入済みタブ → 「適用」→ アプリ再起動で反映。
- 初期ポイントはテスト用に 1000pt（本番では変更予定）。

## 8. バッジとプロフィールカスタマイズ
- 統計タブで解除されたバッジはプロフィールアイコンとして使用可能。
- プロフィール画面で「バッジを変更」→ 5列グリッドから選択。
- 全バッジは絵文字がユニーク。季節/イベント限定バッジは今後追加予定。

## 9. ゲームセンター
- 収録ゲーム：タップチャレンジ / 数当て / スネーク / 2048 / クイズ。
- 結果によって演出レベル（Perfect / Excellent / Good / Normal）。ハイスコア更新時は特別演出。
- ゲーム結果は実績・ポイント・バッジ解除に直結。

## 10. コメント / コミュニティ
- 記事下部でコメント投稿。質の高いコメントでコミュニティ活性化。
- 荒らし防止のため、将来的に通報 & フィルタリング機能追加予定。

## 11. ペット育成ゲーム 🐾

### 基本システム
ニュースを読むほどペットが成長！記事ジャンルに応じた多次元育成が可能。

#### 進化段階
1. **たまご🥚** (レベル 0-9): 初期状態
2. **子供🐣** (レベル 10-29): 最初の進化
3. **成体🦅** (レベル 30-69): 本格的な姿
4. **究極体🔥** (レベル 70-100): 最終形態

#### ジャンル別ステータス
ペットは5つの能力値を持ち、読んだ記事のジャンルで成長：
- **ビジネス💼**: ビジネス・経済記事で成長
- **テクノロジー🖥️**: IT・科学記事で成長
- **エンターテインメント🎬**: 芸能・文化記事で成長
- **スポーツ⚽**: スポーツ記事で成長
- **国際政治🌏**: 政治・国際記事で成長

**育成戦略**:
- **バランス型**: 全ジャンル均等に読んで万能ペット育成
- **特化型**: 1つのジャンルに集中して突出した能力獲得

#### 親密度システム
毎日の世話で親密度レベルがUP（0-100）：
- **知人** (0-20): 基本機能のみ
- **友達** (21-40): 翻訳速度10%UP
- **親友** (41-60): ポイント獲得5%UP
- **相棒** (61-80): ガチャレア率微増
- **運命の仲間** (81-100): 全スキル最大化

**親密度を上げる方法**:
- 記事を読む: +1
- ゲームをプレイ: +2
- 餌をやる（1日1回）: +5
- ペットと遊ぶ: +3

⚠️ **注意**: 3日連続ログインしないと親密度-6

#### ペットの種類
- **ドラゴン🐲**: ビジネス成長+20%、炎エフェクト
- **フェニックス🔥**: テクノロジー成長+20%、蘇生スキル
- **ユニコーン🦄**: エンタメ成長+20%、虹オーラ

ショップで追加ペット購入可能（最大5体まで育成）。

#### ペット図鑑
- 全20種類のペット（基本5 + レア10 + レジェンダリー5）
- 未解除ペットは影で表示
- 解除条件：特定ジャンル100記事読破、実績達成など

### 上級テク
- **好物システム**: 餌やり時にランダムで好物判定（親密度+10）
- **お散歩**: 累計100記事ごとに小報酬（ポイント+50）
- **おしゃべり**: ペットがランダムメッセージ表示（ヒント付き）
- **スキル発動**: 親密度レベルに応じて自動的に恩恵

## 12. FAQ
| 質問 | 回答 |
|------|------|
| 翻訳が表示されない | DeepLキー設定を確認 / ネット接続 / フォールバック辞書が動作します |
| ガチャが引けない | 1日1回制限。日付が変わるまで待機（開発版は制限解除）|
| テーマが変わらない | ショップで適用後アプリを再起動してください |
| 実験実績は? | 統計画面の🧪ボタンで解除（本番非表示予定） |
| ポイント不足 | 実績解除・ガチャ達成・ゲームスコアで獲得可能 |
| バッジをプロフィールに? | プロフィール画面で変更できるアイコンを選択 |
| ペットが進化しない | レベル10/30/70で自動進化。記事を読んで経験値獲得 |
| 親密度が下がった | 3日連続ログインなしで-6。毎日触れ合いが重要 |
| 複数ペット育成は? | ショップで追加ペット購入（1000pt）、最大5体 |

## 13. 成長のコツ / 上級テク

### 効率的な1日の流れ
**朝（5分）**:
- グローバルニュース2-3本 → ペット餌やり → デイリークエスト進捗確認

**昼休み（10分）**:
- 気になる国別ニュース詳細読み → 1ゲームプレイ → お気に入り整理

**夜（15分）**:
- 図鑑未解除のレア実績を狙った行動（長時間読書 / 高難度スコア）
- ペットと遊ぶ（親密度UP） → ガチャ引く → 翌日のデイリークエスト確認

### ポイント効率化
- **レジェンダリー実績優先**: 1回で300pt（コモン30回分）
- **デイリークエスト全達成**: 200pt/日（月6000pt）
- **ペットスキル活用**: 親密度70でポイント+10%常時適用
- **コレクションセット**: 長期目標で追加ボーナス

### ペット育成最適化
- **バランス型**: 毎日異なるジャンルを読む（全5ジャンル）
- **特化型**: 1つのジャンルを集中（レーダーチャート1辺MAX）
- **親密度維持**: 3日以上放置しない、餌やり忘れ防止

### 翻訳学習術
1. 記事を英語原文で流し読み（2分）
2. 日本語訳で詳細理解（5分）
3. 不明単語をメモ（将来: アプリ内単語帳機能）
4. 翌日同じジャンルの記事で復習

## 14. トラブルシューティング
| 症状 | 対処 |
|------|------|
| 実績演出が出ない | 既に解除済みか `AchievementNotifier` のOverlayが過剰。再起動。|
| ガチャ結果が保存されない | `SharedPreferences` 破損→ 再インストール or 設定クリア。|
| フォントが乱れる | 設定で行間/フォントサイズを一度変更→戻す。|
| スクロールが重い | 翻訳キャッシュをクリアし再起動。|
